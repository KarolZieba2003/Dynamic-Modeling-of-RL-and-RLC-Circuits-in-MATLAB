## ðŸš€ Project Overview

This project is a comprehensive analysis of first-order (RL) and second-order (RLC) electrical circuits. It uses the **MATLAB Symbolic Math Toolbox** to analytically solve the ordinary differential equations (ODEs) that govern these circuits.

The primary goal is to understand and visualize the **transient response** of these circuits by analyzing the impact of different parameters, such as initial conditions, input voltage, and component values (L, C).

This was a laboratory assignment for the "Dynamic Systems Modeling" course at the Cracow University of Technology.

## âš¡ Key Features

* **Symbolic Solving:** Uses `dsolve` to find the exact analytical formulas for current $i(t)$ and capacitor voltage $u_c(t)$.
* **Time Domain Analysis:** Generates plots showing the time-based response (e.g., $i(t)$ vs. $t$) for various scenarios.
* **Phase Portrait Analysis:** Generates phase plots (e.g., $i$ vs. $di/dt$) to visualize the system's dynamics and stability.
* **Parameter Analysis:** The scripts are structured to analyze the impact of:
    * Initial Conditions ($i(0)$, $u_c(0)$)
    * Forcing Functions (Input Voltage $u_0$)
    * Physical Components (Varying $L$ or $C$)

## ðŸ”§ How to Use

1.  **Prerequisites:** You must have MATLAB and the **Symbolic Math Toolbox** installed.
2.  **Clone the repository:**
    ```bash
    git clone [https://github.com/TWOJA-NAZWA-UZYTKOWNIKA/TWOJA-NAZWA-REPO.git](https://github.com/TWOJA-NAZWA-UZYTKOWNIKA/TWOJA-NAZWA-REPO.git)
    ```
3.  **Run the script:**
    * Open `CW1.m` (for the RL circuit analysis) or `CW2.m` (for the RLC circuit analysis).
    * Inside the script, **uncomment** the "Case" (Przypadek 1, 2, or 3) you wish to run.
    * Run the script (F5). The script will output the analytical solutions to the console and generate the plots.
